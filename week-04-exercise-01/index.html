Dev Tools Vandalism

I used browser developer tools to temporarily vandalize a website. Below is a screenshot showing my changes. These edits only appeared on my computer and did not affect the actual website.

What I Vandalized

I picked the Wikipedia page for Texas since I currently live here. For the HTML edits, I added a sentence to the intro paragraph that mentions me living in the state and where I am studying. I also deleted the majority of the first paragraph so my edit would stand out more.

For the CSS edits, I changed the background color behind the "Texas" title to cyan so it stood out like a highlighter mark, switched the body paragraph text color to green, and added a red background bar behind the "Jump to content" link at the top of the page. The red background bar was a complete accident as I explored the CSS and was trying to see what changes would stand out.

I've tinkered with the browser developer tools before while at work. It is especially helpful in narrowing down UI bugs.
